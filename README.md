# VoiceOut - Web

## Run Locally

Clone the project

```bash
  git clone https://github.com/Quixotek/VoiceOut-Web.git
```

Go to the project directory

```bash
  cd VoiceOut-Web
```

Install dependencies

```bash
  yarn install
```

Start the server

```bash
  yarn dev
```

## Running Tests

To run tests, run the following command

```bash
  yarn test
```

## Running Codegen

To run codegen, run the following command

```bash
  yarn generate
```
